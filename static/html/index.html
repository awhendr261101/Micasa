Users

HTTP Methods Path Description
GET /users This endpoint retrieves all users.
GET /users/:id This endpoint retrieves one user, specified by the id.
POST /users/register This endpoint is used to register a user.
PATCH /users/:id This endpoint is used to update a user, specified by the id.
POST /users/login This endpoint is used to login a user.
DELETE /users/:id This endpoint is used to delete a user, specified by the id.
